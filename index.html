Política de Privacidad para "Cronómetro para Casos" Última actualización: 03 de Octubre de 2025 Esta Política de Privacidad describe cómo la extensión de Chrome "Cronómetro para Casos" (en adelante, "la Extensión") maneja la información. Nuestra filosofía es simple: solo utilizamos los datos estrictamente necesarios para que la extensión funcione como esperas. No recopilamos, almacenamos, ni compartimos tu información personal. 1. ¿Qué información maneja la Extensión? La Extensión maneja dos tipos de información, ninguno de los cuales es personal: Datos de Configuración del Usuario: Para ofrecer una experiencia personalizada, guardamos tus preferencias directamente en tu navegador usando la API chrome.storage. Esto incluye: El tiempo en minutos que configuras para la alarma. Si decides activar o desactivar las notificaciones. Estos datos se almacenan localmente en tu perfil de Chrome y no son accesibles para nosotros ni para terceros. Datos Temporales de la Página Web: Para funcionar, la extensión necesita leer el contenido de la página crm.adminml.com. Identifica elementos como el título de un caso para mostrarlo en las notificaciones de alarma (por ejemplo, "¡Tiempo excedido en el caso 'XYZ'!"). Esta información se usa únicamente en tiempo real para mostrar la alerta y nunca se guarda, almacena o transmite fuera de tu navegador. 2. Permisos Requeridos en Chrome Solicitamos los siguientes permisos y esta es la razón: storage: Para guardar tus configuraciones (el tiempo de alarma y la preferencia de notificación) como se describe arriba. host_permissions sobre https://crm.adminml.com/*: Para que la extensión pueda ejecutarse en el sitio web del CRM, añadir los cronómetros visuales y mostrar las alertas. La extensión no funcionará en ningún otro sitio web. 3. ¿Compartimos tu información? No. Nunca. Toda la lógica de la extensión se ejecuta localmente en tu navegador. No tenemos servidores y no recopilamos ninguna analítica, dato personal o de navegación. 4. Cambios a esta Política Podemos actualizar esta política en el futuro si añadimos nuevas funcionalidades. Te notificaremos de cualquier cambio importante actualizando la extensión y su descripción en la Chrome Web Store. 5.
